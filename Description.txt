SearhEngine - приложение для поиска с использованием API, таких поисковиков как Google,Bing,Yandex.
Структура проекта:
SearchEngine.Core - функционал поиска
SearchEngine.Domain - сущности БД
SearchEngine.WebApp - ASP.NET CORE MVC приложение
SearchEngine.Tests - тесты 
Основной функционал - выдача первого результата поиска в виде 10 значений и занесение его в БД.
Если результат полученный первым имеет флаг ошибки, то такой результат игнорируется и берется следующий положительный результат.
Поиск по БД - по умолчанию возвращаются все запросы из бд. Фильтр применяется по данным запроса.
Для создания БД имеются миграции в SearchEngine.Domain
Данные запроса хранятся в 2 таблицах. Результаты значений заносятся в таблицу Results в виде Json. При работе с объектами с использованием ef Json конвертируется в коллекцию.
Конфигурации - позволяют изменять данные поисковиков "на лету". т.е. данные при запуске сервиса в файл с найстройками не заносятся , а используются в текущем режиме в объекте IConfiguration.
Для полного изменения данных API поисковиков, нужно вносить данные в файл appSettings.json в секцию EnginesConfig по имени поисковика. 
При добавлении нового функционала поиска необходимо реализовать интерфейс ISearchEngine и при необходимости(если опции API отличаются от класса SearchEngineOptions) создать класс унаследованный от SearchEngineOptions. 
Внести данные доступа в appSettings.json. Добавить определение поисковика в класс SearchEngineFactory.
